🎯 개발 배경 및 목표

개발 배경 (문제점 및 필요성)

대학생들이 시험 준비를 위해 PDF 교재나 노트를 읽고 분석하는 데 많은 시간이 소요되고, 필요한 정보를 효율적으로 찾는 데 어려움이 있습니다. 학생들이 PDF 자료에서 빠르게 정보를 검색하고, 질문에 대해 즉시 답변을 받을 수 있는 도구가 필요합니다.

주요 목표

  • PDF Q&A 챗봇 구축 (PDF 업로드 및 질문 기능 제공)
  • AI 모델을 활용한 정보 추출 및 실시간 답변 생성
  • 정확한 답변을 제공하여 시험 대비 학습 도우미 역할 수행

💡 솔루션: RAG (Retrieval-Augmented Generation)

사용자가 자신의 시험 자료(PDF)를 업로드하면, 챗봇은 자료를 분석/벡터화하여 저장합니다. 사용자가 질문하면, 관련성 높은 부분을 검색(Retrieval)한 후, LLM을 통해 정확한 답변을 생성(Generation)합니다.

✨ 주요 기능

📤

업로드 및 자동 색인

다수의 파일을 안전하게 업로드하고, 텍스트 추출, 청킹, 벡터 임베딩을 자동으로 수행하여 DB에 저장합니다.

🤖

RAG 기반 QA 시스템

사용자 질문 의도를 파악하고, 벡터 검색을 통해 추출된 관련성 높은 텍스트를 LLM에 전달하여 정확한 답변을 생성합니다.

📚

신뢰도 높은 출처 인용

생성된 답변과 함께 원본 PDF 파일명, 페이지 번호, 원문 문장을 제시하여 답변의 신뢰성을 높입니다.

🖥️

사용자 친화적 UI

직관적인 파일 관리 및 실시간 채팅 기능을 갖춘 반응형 웹 애플리케이션을 제공합니다.

🚀 파급 효과 및 향후 계획

📈 개발 시 파급 효과

학생들에게

시험 준비 시 필요한 정보를 빠르게 얻을 수 있어 학습 효율성이 크게 향상됩니다. 또한, 시간 절약을 통해 더 많은 학습을 할 수 있게 됩니다.

교육 시스템에

디지털 학습 자료의 효율적인 활용을 촉진하고, AI 기술을 통한 새로운 학습 경험을 제공하여 교육 시스템에 혁신적인 변화를 가져올 수 있습니다.

📅 향후 계획

MVP(Minimum Viable Product) 개발 완료 후, 학습 능률을 더욱 높여줄 수 있는 추가 기능을 구현할 예정입니다:

  • 강의 자료 요약 기능: 긴 문서를 빠르게 핵심만 파악할 수 있도록 돕습니다.
  • 플래시 카드 생성 기능: 시험 대비를 위한 자동 암기 카드를 생성합니다.
